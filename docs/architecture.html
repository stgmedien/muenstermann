5. Audit-Trail

flowchart LR
    A["Anwendung
SET LOCAL app.user_id = '...'"]
    T["fachliche Tabelle
z.B. catalog.cleaning_agent"]
    L["audit.activity_log
(actor, action, old_row, new_row, txid)"]

    A -- "UPDATE / INSERT / DELETE" --> T
    T -- "Trigger" --> L

    style L fill:#fff3cd,stroke:#f0ad4e

Rechtssichere Nachvollziehbarkeit für HACCP, GefStoffV, IFS-Audit. Aktor wird per Session-Variable gesetzt; in Phase 1 macht das jedes ETL-Skript, später kommt es aus der Auth-Schicht der Anwendung.
